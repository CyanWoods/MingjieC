# MingjieC

明解C语言代码、习题练习（基于unix环境）

### 注意：使用POSIX标准的编译器会定义宏CLOCKS_PER_SEC的值为1000000,也就是1000^2，而作者所用的编译器认为CLOCKS_PER_SEC为1000，这就导致作者的很多示例代码在UNIX-like的机子上是无法得到正常的结果。

### 由于macOS和Windows对于time()函数的区别（具体看这里：https://zhuanlan.zhihu.com/p/349949616） ，书上面所有用time()函数计算程序运行时间的代码实际计算的都是cpu时间而非挂墙时间，解决方式上面的链接也提供了，等我在第二次回顾这些练习代码的时候应该会改正。

具体想法思路应该会写在每一个练习之后的readme里面（现在还没有）
